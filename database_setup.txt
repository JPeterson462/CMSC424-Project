1. Install PostgreSQL 10 (latest version)
    - installer: https://www.enterprisedb.com/downloads/postgres-postgresql-downloads#windows

2. Set up psycopg2
    - run from the command prompt:
        pip install psycopg2

3. Add PostgresSQL to your PATH environment variable
    - open cmd.exe and run:
        setx PATH "%PATH%;C:\Program Files\PostgreSQL\10\bin"

4. Create a new database either through pgAdmin4 or by executing this from psql:
    CREATE DATABASE testdb OWNER postgres

5. Update settings.py in the Django app to have:
    DATABASES = {
        'default': {
            'ENGINE': 'django.db.backends.postgresql',
            'NAME': 'testdb',
            'USER': 'postgres',
            'HOST': 'localhost',
            'PASSWORD': 'password',
        }
    }

6. Apply the migrations for the new database
    python manage.py migrate
